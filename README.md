# Steam Scholarship Challenges
This repo contains a set of problems that candidates for the STEAM scholarship
must solve!

They are available in the following programming languages

- Java
- Kotlin
- C
- C++
- JavaScript
- TypeScript
- Julia
- Rust
- Go
- Haskell
- Python
- Ruby
- Lua
- R
- ASM (x86_64)
- Perl
- Pascal
- Fortran
- Nim
- Zig
